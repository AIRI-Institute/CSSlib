# CCSlib

CCSlib is an open-source code for building composition/configuration spaces (CCS) of disordered crystals.

Installation
-----
**CCSlib** requires **Supercell** program. Details on **Supercell** installation can be found at the corresponding [website](https://orex.github.io/supercell/download/).

Tutorial
-----
The best way to learn how to use **CCSlib** is through the [tutorial notebook](ccslib_example.ipynb).
